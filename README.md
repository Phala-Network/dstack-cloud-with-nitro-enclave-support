## dstack KMS Documentation

This repository contains documentation for the dstack-based KMS deployment and operations project.

The documents are organized under the `docs/` directory and cover:

- Overall architecture and threat model
- GCP environment and CVM setup
- KMS image build and deployment
- Nitro Enclave integration and RA-TLS
- Governance and on-chain integration
- Monitoring, operations, and runbooks
- Testing, validation, and final release notes

Start with `docs/00-documentation-plan.md` for an overview of all documents and their purpose.
